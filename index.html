PyPDF2 & pdfplumber (Extra√ß√£o de PDF)
Ferramentas para extrair texto, metadados e tabelas de documentos PDF. **PyPDF2** √© ideal para texto e manipula√ß√£o de p√°ginas, enquanto **pdfplumber** √© excelente para tabelas e texto com posi√ß√µes.

import PyPDF2
import pdfplumber

# Exemplo com PyPDF2 (texto simples)
def extract_text_pypdf2(pdf_path):
    text = ""
    with open(pdf_path, 'rb') as file:
        reader = PyPDF2.PdfReader(file)
        for page in reader.pages:
            text += page.extract_text()
    return text

# Exemplo com pdfplumber (texto e tabelas)
def extract_data_pdfplumber(pdf_path):
    extracted_text = ""
    extracted_tables = []
    with pdfplumber.open(pdf_path) as pdf:
        for page in pdf.pages:
            extracted_text += page.extract_text() + "\n"
            tables = page.extract_tables()
            if tables:
                extracted_tables.extend(tables)
    return extracted_text, extracted_tables

pdf_text_pypdf2 = extract_text_pypdf2('documento.pdf')

# Especifique o caminho se necess√°rio (ex: Windows):

def extract_text_from_scanned_pdf(pdf_path):
    text = ""
    pdf_document = fitz.open(pdf_path)
    for page_num in range(len(pdf_document)):
        page = pdf_document.load_page(page_num)
        pix = page.get_pixmap()
        img = Image.open(io.BytesIO(pix.pil_tobytes(format="png")))
        text += pytesseract.image_to_string(img, lang='por') # 'por' para portugu√™s
    return text

scanned_pdf_content = extract_text_from_scanned_pdf('documento_digitalizado.pdf')

Dicas Essenciais para An√°lise de Dados

üöÄ Otimiza√ß√£o de Performance

Use Pandas de forma eficiente: Prefira opera√ß√µes vetorizadas ao inv√©s de loops. Use df.apply() ao inv√©s de for loops quando poss√≠vel.
Gerencie mem√≥ria: Use df.dtypes para verificar tipos de dados e pd.to_numeric() com downcast para otimizar o uso de mem√≥ria.
Para grandes datasets: Considere usar Dask ou Polars ao inv√©s do Pandas tradicional para melhor performance.

üßπ Limpeza e Prepara√ß√£o de Dados

Sempre explore primeiro: Use df.info(), df.describe() e df.head() para entender seus dados antes de come√ßar a an√°lise.
Trate dados faltantes: Use missingno para visualizar padr√µes de dados ausentes e escolha a estrat√©gia apropriada (remo√ß√£o, imputa√ß√£o, etc.).
Valida√ß√£o de dados: Sempre valide a qualidade dos seus dados ap√≥s transforma√ß√µes usando assert statements.

üìä Visualiza√ß√£o Eficaz

Escolha o gr√°fico certo: Barras para categorias, linhas para s√©ries temporais, scatter para correla√ß√µes, histogramas para distribui√ß√µes.
Use cores com prop√≥sito: Aplique cores para destacar insights, n√£o apenas por est√©tica. Considere daltonismo na escolha de paletas.
Interatividade: Use Plotly para dashboards e apresenta√ß√µes, mas Matplotlib/Seaborn para an√°lises explorat√≥rias r√°pidas.

ü§ñ Machine Learning

Valida√ß√£o cruzada: Sempre use cross_val_score ou KFold para avaliar modelos de forma robusta.
Feature engineering: Invista tempo criando features relevantes. Use category_encoders para vari√°veis categ√≥ricas.
Baseline simples: Comece sempre com modelos simples (regress√£o linear, √°rvore de decis√£o) antes de partir para algoritmos complexos.

üîß Melhores Pr√°ticas

Ambiente virtual: Sempre use venv ou conda para isolar depend√™ncias dos seus projetos.
Documenta√ß√£o: Use docstrings nas suas fun√ß√µes e mantenha um README.md atualizado em seus projetos.
Controle de vers√£o: Use Git para versionar c√≥digo e DVC (Data Version Control) para versionar datasets.
Notebooks organizados: Use c√©lulas markdown para explicar seu racioc√≠nio e mantenha c√≥digo limpo e comentado.
